    • insertAfter

      Declaration

      LinkedListElement!T* insertAfter(LinkedListElement!T* element, T v);

      Insert value v after element.

      Return Value

      Pointer to inserted element.

      Note: element must be not null.

      Examples

      1. LinkedList!int list; scope(exit) list.free(); auto first = list.append(1); auto last = list.append(2); list.insertAfter(first, 3); assert(list.length == 3); auto arr = list.toArray(); assert(arr == [1,3,2]); Delete(arr);

    • removeFront

      Remove the first element.

      Note: list must be non-empty.

      Examples

      1. LinkedList!int list; scope(exit) list.free(); list.insertBack(0); list.removeFront(); assert(list.length == 0); list.insertBack(1); list.insertBack(2); list.insertBack(3); list.removeFront(); assert(list.length == 2); import std.algorithm : equal; assert(equal(list.byElement(), [2,3]));

    • find

      Declaration

      LinkedListElement!T* find(T v);

      Search for element with value v.

      Return Value

      Found element or null if could not find.

      Examples

      1. LinkedList!int list; scope(exit) list.free(); assert(list.find(42) is null); list.insertBack(13); list.insertBack(42); auto first = list.find(13); assert(first && first.datum == 13); auto second = list.find(42); assert(second && second.datum == 42); assert(list.find(0) is null);
